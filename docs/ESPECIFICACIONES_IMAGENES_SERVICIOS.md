# Especificaciones e imágenes para servicios s1–s14

## Especificaciones técnicas (todas las imágenes)

| Parámetro | Valor |
|-----------|--------|
| **Dimensiones mínimas** | 1080 × 720 px (ratio 3:2) |
| **Recomendado** | 1620 × 1080 px o 1920 × 1280 px |
| **Formato** | JPG, 90–95% calidad |
| **Ratio** | 3:2 (horizontal) |
| **Espacio para overlay** | Zona inferior ±25% reservada para gradiente oscuro (número/icono) |
| **Uso** | Thumbnail (360×240) y detalle (540×360) en web |

### Estilo visual (unificado para s1–s14)

- **Idioma:** Todas las imágenes deben ser en español (elementos visuales, texto si lo hubiera, contexto corporativo hispanohablante)
- **Tono:** Profesional, corporate, consultoría boutique
- **Paleta:** Teal #216a69, tonos neutros, blancos/grises
- **Composición:** Imágenes conceptuales o de contexto laboral (no stock genérico)
- **Evitar:** Caras reconocibles, logotipos de marcas, elementos muy literales
- **Preferir:** Metáforas visuales, datos, equipos, tecnología, estrategia

## Resumen para generadores de imágenes (Midjourney, DALL-E, etc.)

**Regla explícita:** Cada prompt genera exactamente **una imagen**. 1 solicitud = 1 imagen. La aplicación debe producir una única imagen por cada prompt ingresado; no variaciones ni múltiples resultados.

**Todas las imágenes deben ser en español.** Los prompts están en español para que el resultado visual refleje contexto corporativo hispanohablante. Si el generador requiere inglés, traduce el prompt manteniendo las instrucciones explícitas: "generate exactly one image" y "output image in Spanish / imagen en español".

Para cada imagen, añadir estos parámetros de estilo:

```
--ar 3:2 --style raw --v 6
```

O equivalente según la herramienta:
- **Aspect ratio:** 3:2
- **Estilo:** Fotografía o ilustración realista, corporativa
- **Excluir:** rostros reconocibles, marcas, texto legible
